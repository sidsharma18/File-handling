# File Handling CRUD Operations in Python

This project demonstrates how to perform basic CRUD (Create, Read, Update, Delete) operations using Python file handling.  
It is a simple, menu-driven program designed for beginners to understand how data can be stored and managed using text files without a database.

---

## Features
- Create new files and write data
- Read existing file content
- Update file data
- Delete files
- List available files and folders
- Beginner-friendly Python code

---

## Tech Stack
- Python
- File Handling (Text Files)
- pathlib module
